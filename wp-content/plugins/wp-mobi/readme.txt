== Installation ==
Follow the following steps to install a plugin or write to us at <a href="mailto:contact@wpandro.com">contact@wpandro.com</a> We will set it up for you.
  
1. Upload WpMobi to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. New menu named as WpMobi will be visible in your WordPress admin panel.
4. Login to your WpAndro.com account and visit Dashboard.
5. Optin for WpAndro Service( <a href="https://wpandro.com/how-to-obtain-app-key-and-secret-key-for-wpmobi/">Read More</a> ).
6. Copy and Paste your App and Secret Key into WpMobi General Settings.
7. Now visit Build And Publish Tab.
8. Click on the Build to create application for your WordPress website.
9. Build a application may take 24 Hours. Once you revisit you can download and publish your mobile application.
